# Accessibility Rules (ATE9)

## 基本
- 見出し階層は `h1` → `h2` → `h3` を崩さない。
- ボタンとリンクは semantic なタグを使う（`<button>`, `<a>`）。

## 画像
- 装飾目的の画像は `aria-hidden="true"` または空の `alt`。
- 意味のある画像には簡潔な `alt` を必ず付ける。

## キーボード操作
- 主要なボタンやリンクはキーボード操作でフォーカスできることを前提に実装。
- 独自コンポーネントを作る場合は `role` と `aria-*` を検討する。

## コントラスト
- テキストと背景のコントラスト比は WCAG AA 以上を目安に。
- Electric Blue 上の白文字はフォントサイズを十分大きくする。
